# Design Skills

Anti-AI-slop design and audit skills installed in `.agents/skills/`:

- **hallmark** — invoke with `/hallmark`, `/hallmark audit`, or `/hallmark redesign`. Pattern library: named component genres, macrostructures, motion, typography.
- **design-taste-frontend** — invoke with `/tasteskill` or `/design-taste`. Three Dials system (DESIGN_VARIANCE, MOTION_INTENSITY, VISUAL_DENSITY), Brief Inference, Pre-Flight checklist, GSAP skeletons.
- **savage-visual** — invoke with `/savage-visual [routes]`. Brutal evidence-first visual audit: captures every route at 320/390/768 px, judges pixels against 12 hard gates, KILL/FIX/POLISH verdicts. No screenshot, no finding.
- **flow-autopsy** — invoke with `/flow-autopsy [journeys]`. User-centricity audit: drives real journeys on video, scores taps-to-value, dead ends, intent loss, jargon. The recording is the report.
- **emil-design-eng** — invoke with `/emil-design-eng`. Emil Kowalski's UI polish and animation philosophy.

Use hallmark for component patterns and genre selection. Use design-taste-frontend for enforcing quality standards and tuning visual intensity. Both complement each other.
